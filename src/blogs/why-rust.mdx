Someone recently asked me why I like Rust so much. I thought I'd write a short post about it. If you don't know this yet, Rust has been consistently voted the most loved programming language since 2016 in the StackOverflow Developer Survey. So, why is that? What makes Rust so special?

### What is Rust?

Rust is a systems programming language that started out as a personal project of one Graydon Hoare who was working at Mozilla then. How cool is that? The most loved programming language today was once a side project of an an amazing programmer.
